Stubz Documentation

A guideline to Stubz' endpoints. This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL & Query Parameters HTTP Method Request format Response format
Return a list of all movies /movies GET None A JSON object holding all movies along with their info
Return data about a single movie by title /movies/[title] or /movies/:title GET None A JSON object holding data about a movie:
{
"id": 1,
"imageUrl": "",
"title": "Interstellar",
"year": "2014",
"genre": "Sci-fi",
"director": { name: 'Christopher Nolan', bio: 'British-American', birth: 'July 30, 1970', death: '-'}
}
Return movies according to the genre /movies/?genre=[genre] or /movies/?genre=:genre GET The search query contains a genre type A JSON object holding all movies that share the genre
Return data about a director by name /directors/[name] or /directors/:name GET None A JSON object holding data about the director and associated movie(s)
Allow new users to register /users POST A JSON object holding data about the new user to add:
{
"username": "John Smith",
"first_name": "John",
"last_name": "Smith",
"password": "ilovemovies123",
"email": "johnsmith@gmail.com"
} 		A JSON object holding data about the new user that has been successfully registered:
{
"username": "JohnSmith.2021",
"first_name": "John",
"last_name": "Smith",
"password": "ilovemovies123",
"email": "johnsmith@gmail.com",
"user_id: [uuid.v4()]
}
Allow users to view their profile /users/[username] or /users/:username GET None A JSON object holding all data about a specific user:
{
"user_id": [uuid.v4()],
"username": "careerfoundry",
"first_name": "Career",
"last_name": "Foundry",
"password": "moviesapp",
"email": "careerfoundry@gmail.com",
"favorites": ["Fury"]
}
Allow users to update their info /users/[user_id] or /users/:id PUT A JSON object holding the update about the user:
{
"email": "careerfoundry20@gmail.com"
} 		>A verification text stating whether or not the update has been successful
Allow users to add movies to their list of favorites /users/[user_id]/favorites/[movie] or /users/:id/favorites/:movie POST Registered user individual (user_id) adds a movie to their list of favorites A verification text stating whether or not the movie has been successfully added
Allow users to remove movies from their list of favorites /users/[user_id]/favorites/[movie] or /users/:id/favorites/:movie DELETE Registered user individual (user_id) removes a movie from their list of favorites A verification text stating whether or not the movie has been successfully deleted
Allow existing users to deregister /users/[user_id] or /users/:id DELETE None A verification message stating whether or not the user was able to successfully deregister
